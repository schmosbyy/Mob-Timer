<!-- Plugin description -->
A <strong>collaborative timer</strong> for your mob team!
<br>
<strong>Usage Instructions:</strong>
<ol>
  <li>Open the Mob Timer Plugin Tool windows by clicking the Clock Icon with the play button.</li>
  <li>Configure the Mob Timer by clicking "Your Session" -> "Show Member Form" : This will allow you to add all members of your team.</li>
  <li>Once you have added the team members and maybe the Goals for the day. Click the Icon on the left sidebar for the "PlayPauseTimer" in the Tool Window</li>
  <li>This will allow the team to Mob out the goals and use the PlayPauseTimer to toggle the timer while mobbing.</li>
</ol>
<!-- Plugin description end -->

## Installation

- Using the IDE built-in plugin system:
  
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "Mob-Timer"</kbd> >
  <kbd>Install</kbd>
  
- Using JetBrains Marketplace:

  Go to [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/MARKETPLACE_ID) and install it by clicking the <kbd>Install to ...</kbd> button in case your IDE is running.

  You can also download the [latest release](https://plugins.jetbrains.com/plugin/MARKETPLACE_ID/versions) from JetBrains Marketplace and install it manually using
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>

- Manually:

  Download the [latest release](https://github.com/schmosbyy/Mob-Timer/releases/latest) and install it manually using
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>
